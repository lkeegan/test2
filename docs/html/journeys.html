async journeys.views.list(request, *args, **kwargs)

Journey list

Returns a list of current process states of the patient journey by patient id.

  • state: Number of the patient state

  • sub_id: Returns whether the current state is a decision step (1, otherwise 0)

  • completed: Date of the completed process state

Args:

request: Request

Returns:

any: JSON response
