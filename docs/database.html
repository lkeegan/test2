database.views

database.views.data_dir = '/home/lkeegan/SSC/systemplatform/backend/data'

Parses the JSON files in the /data/* directories as FHIR resources using fhirpy and adds the initial resource data to the FHIR server.

database.views.get_files(dir)

Returns a list of all json file paths from the directory

data_dir: Base path of the data subdirectory

Args:

dir (any): Directory

Returns:

any: path to json file

database.views.get_patient_state(value)

Returns the formatted patient process state

Args:

value (int): Value of the patient state (whole numbers from 1 to 9, including decision steps)

Returns:

int: Value of the patient state which is represented as a decimal number from 1 to 6 (decisions are mapped to fractional numbers)

database.views.index(request)

Example FHIR JSON data files which are stored in the /data folder. FHIR resources are initialized at the /database route.

Args:

request (Any): HTTP Request

Returns:

any: HTML view (database_form)

database.views.meta_tag(resource)
database.views.pat_resource_extensions(resource)

Adds the following information to the Patient resource via extensions:

  • ids of available questionnaires

  • the state of the patient path

  • check to enable selection of questionnaires

  • path_ext: Patient path (state)

  • quest_extensions: default selected questionnaires (ID)

  • act_ext: questionnaire selection action required

Args:

resource (any): Patient resource
