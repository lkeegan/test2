database.views.pat_resource_extensions(resource)

Adds the following information to the Patient resource via extensions:

  • ids of available questionnaires

  • the state of the patient path

  • check to enable selection of questionnaires

  • path_ext: Patient path (state)

  • quest_extensions: default selected questionnaires (ID)

  • act_ext: questionnaire selection action required

Args:

resource (any): Patient resource
