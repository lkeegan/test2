Development

Basic info on cloning the repo, installing dependencies, etc.

Run locally with Python

Create the database

python manage.py migrate

Create a new Django account

python manage.py createsuperuser

Start the Django server

python manage.py runserver

Start the HAPI FHIR server using Docker docker run -p 8080:8080 hapiproject/hapi:latest

Initialize the server with FHIR resources

  • By opening the /api/database route at http://127.0.0.1:8000/api/database/

Run with Docker

Build and run the Docker containers

docker-compose up --build -d
